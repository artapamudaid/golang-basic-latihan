# Golang Basic Learning

Proyek belajar dasar-dasar bahasa Go.

## Struktur

- `main.go` — Entry point aplikasi
- `latihan1.go` — Latihan pertama: variabel, kondisi, perulangan, dan fungsi

## Cara Menjalankan

```bash
go run .
```

## Materi yang Dipelajari

- Variabel dan tipe data
- Percabangan (`if-else`)
- Perulangan (`for`)
- Fungsi
